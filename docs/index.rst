Welcome to Hodu's documentation!
===================================

**Hodu** is a Python library for cooks and food lover doggy's
that creates pet food recipes mixing random ingredients
It pulls data from the `The spruce pets database <https://www.thesprucepets.com/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
